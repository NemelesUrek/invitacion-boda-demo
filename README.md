# Invitación de boda — Demo interactiva

Landing page de invitación de boda, **archivo único** (HTML + CSS + JS), responsiva y sin dependencias de imágenes externas.

## Características
- 🎨 **4 paletas** intercambiables en vivo (Salvia · Blush · Marino · Mocha)
- 🌗 **Modo claro / oscuro** con persistencia (localStorage)
- ⏳ Contador regresivo al día de la boda
- 📝 Formulario RSVP interactivo (demo front-end)
- 6 secciones: Invitación · Nuestra Historia · Código de Etiqueta · RSVP · Itinerario · Información
- Visuales 100% CSS/SVG · Bootstrap 5 + Google Fonts

> **Demo** con datos ficticios. Desarrollada por **Nemeles**.
